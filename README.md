# Automated-Clinician-Patient-Interaction-Analyzer
Automated Clinician-Patient Interaction Analyzer is an AI-powered system that transcribes Zoom-based medical consultations in real time and uses LLMs to map clinician questions with patient responses. It generates intelligent summaries and extracts follow-up actions, helping reduce manual documentation and improve clinical efficiency.


Real-time AI Transcription & Mapping
Developed a system to transcribe live Zoom consultations and intelligently align clinician questions with patient responses using LLMs.
Automated Clinical Summaries & Task Extraction
Generated concise summaries and actionable follow-ups, reducing manual documentation workload by ~60%.
Scalable Hybrid Backend Architecture
Built using FastAPI for AI/LLM workflows and Node.js + WebSockets for real-time streaming and orchestration.
Interactive Angular Dashboard
Designed a dashboard to visualize live transcripts, extracted insights, and action items, enhancing clinical decision-making.
